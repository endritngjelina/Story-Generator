# 📖 Story Generator

A fun, interactive web-based story generator that creates unique stories 
with randomized elements — built with pure HTML, CSS, and JavaScript.

## 🚀 Features

- Generate creative and unique stories instantly
- Multiple versions with different story templates and styles
- Simple, browser-based — no installation required
- Lightweight and fast with no external dependencies

## 📁 Project Structure

| File | Description |
|---|---|
| `story generator 0.html` | Base version — core story generation logic |
| `story generator 1.html` | Version 1 — expanded story elements |
| `story generator 2.html` | Version 2 — updated UI and new story types |
| `story generator 3.html` | Version 3 — additional randomization features |
| `story generator 4.html` | Version 4 — improved styling and interactivity |
| `story generator 5.html` | Version 5 — latest and most complete version |

## 🛠️ How to Use

1. Clone or download the repository
   ```bash
   git clone https://github.com/endritngjelina/Story-Generator.git
   ```
2. Open any `.html` file in your browser
3. Click the generate button to create a new story
4. Refresh or click again for a brand new story every time!

## 💻 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
